# Living Field — Full Next-Round Direction for Claude

This document is intentionally aggressive and comprehensive. It combines:
- direct strike corrections
- expanded art direction
- engineering guidance
- implementation requirements
- tuning guidance
- acceptance criteria

The goal is to get the project from **promising atmospheric SVG system** to **a convincing alien geometric field**.

---

## Executive Summary

The current version is better than the first pass, but it is still too dependent on:
- circles
- orbital/ring grammar
- readable symbolic fragments
- placement over emergence
- layering over true field behavior

It is at risk of becoming **beautiful sci-fi UI residue**.

That is not the target.

The target is a fullscreen generative field that feels like:

- alien geometric weather
- pressure systems made visible
- living symbolic ecologies
- partially emergent structures
- regional dialects of shape and motion
- elegant but uncanny mathematics

The next pass must move from:
**glyphs in a dark field**
to
**strange organisms and pressures arising from a shared field logic**.

---

# Part I — Direct Strikes

These are the highest-priority corrections.

## Strike 1 — Break the circle addiction

### Problem
The system is still too ring/circle/orbit dependent.

### Action
Treat perfect circles as exceptional events, not a default primitive outcome.

### Requirements
- hard-cap the proportion of motifs containing full circles
- strongly prefer broken circles over perfect circles
- strongly prefer interrupted arcs over complete rings
- if a circle is used, it must justify itself compositionally

### Rule
A perfect circle should feel rare enough that the viewer notices it.

### Implementation
- add shape-family weighting that penalizes perfect closure
- add a global `closureBias` or `perfectClosurePenalty`
- add per-region closure preferences
- add post-generation rejection for motifs that drift too close to “default orbital icon”

---

## Strike 2 — Replace orbital grammar with tension grammar

### Problem
The current motifs still imply:
- orbital interfaces
- instrument dials
- HUD remnants
- symbolic widgets

### Action
Shift from orbit/ring language to tension/deformation language.

### Prefer forms that feel:
- biased
- pulled
- bent
- punctured
- interrupted
- partially scaffolded
- asymmetrically enclosed
- mid-reorganization
- stretched by invisible force

### Required motif characteristics
- offset mass
- nonuniform spacing
- incomplete closure
- directional bias
- visible imbalance
- internal tension marks
- one dominant side or arm
- fragments implying a whole without completing it

---

## Strike 3 — Make macro forms environmental, not symbolic

### Problem
Large ghost structures currently read as big dim interface fragments.

### Action
Macro forms must feel like environmental field pressures, not giant faded symbols.

### Requirements
Macro forms should resemble:
- warped contour veils
- pressure bands
- bent manifolds
- partial shell fields
- asymmetric environmental arcs
- diffuse interference fronts

### They should not resemble:
- giant faded dials
- UI rings
- dashboard gauges
- enlarged normal motifs

### Implementation
- create a separate macro-form generator, not just “oversized motif instances”
- use softer edges, longer spans, less literal closure
- introduce broad low-frequency deformation
- allow offscreen anchors and incomplete entry/exit into frame

---

## Strike 4 — Show real field behavior

### Problem
The screenshots show layering, but not enough evidence of actual eddies, pockets, lanes, or basins.

### Action
Make field dynamics visually legible.

### The viewer should be able to perceive:
- drift corridors
- rotational neighborhoods
- converging or diverging tendencies
- calm regions
- local synchrony
- delayed response to invisible forces

### Implementation
- tie morphology more strongly to flow sampling
- allow local tangent alignment
- allow group response to shared attractors
- add temporal inertia so changes lag rather than snapping
- increase visible regional coherence slightly

---

## Strike 5 — Make motifs emerge from the field, not sit inside it

### Problem
Many motifs still feel placed rather than grown.

### Action
Motifs should feel like field outcomes.

### Requirements
- nearby motifs should visibly share deformation tendencies
- motifs in a region should inherit regional morphology bias
- local clusters should share motion character, not just proximity
- accent motifs should feel like condensed events from the same field, not decorative overlays

# Part II — Flood the Zone

This is the broader instruction set. Use it to reshape the system deeply.

## 1. Core Visual Philosophy

The Living Field should feel like:
- an ecology, not a layout
- a pressure map, not a gallery
- a shared language, not a bag of symbols
- authored emergence, not random decoration
- beautiful, but not familiar
- coherent, but not fully decodable

### Avoid
- generic procedural SVG wallpaper
- default sci-fi UI aesthetics
- circles as comfort objects
- tidy symmetry
- uniform icon legibility
- evenly distributed decorative motifs

### Prefer
- asymmetry
- incompletion
- pressure
- morphic ambiguity
- local kinship
- atmospheric depth
- rare punctuation events
- regional personality

---

## 2. Geometry Language Requirements

The next pass needs a substantially richer geometry vocabulary.

### Allowed ingredients
Circles, arcs, nodes, and lines can still exist, but only as ingredients.

### Not acceptable as dominant output
- perfect circles
- concentric rings
- balanced targets
- clean radial spokes
- generic orbit widgets
- evenly symmetric shells
- standard halo shapes

### Required dominant qualities
Most motifs should contain several of the following:
- incomplete closure
- off-axis anchor
- one-sided branching
- asymmetric enclosure
- puncture point
- split lobe
- ribbed or segmented internal spine
- counterweighted satellite cluster
- broken manifold
- skewed shell
- nonuniform petal remains
- interrupted tangential sweep

### Motif families to add
At minimum, add strong variants of these:

#### A. Broken crescent families
- crescents with offset nicks
- crescents with internal tension bars
- crescents with trailing debris
- crescents that almost close but do not

#### B. Split-lobe families
- two-lobe forms that never resolve symmetrically
- folded shell pairs
- offset lobe + puncture structures
- lobe clusters pulled along one axis

#### C. Ribbed spine families
- dominant stem/spine with small offset ribs
- branch-biased scaffolds
- long biased ligatures
- spine with clustered satellites

#### D. Interrupted shell families
- partial enclosures with gap emphasis
- layered shell fragments
- shells with puncture marks or detached nodes
- half-shells bending along flow lines

#### E. Knot/manifold families
- tangled partial loops without complete closure
- warped crossings
- near-knots with missing closures
- manifolds that imply continuity but fragment instead

#### F. Pressure-struck fragments
- fragments compressed by local field
- repeated biased arc segments
- broken contour residues
- stretch-fractured motifs

#### G. Semi-biological scaffolds
- elegant unknown-tool or organism-like shapes
- asymmetrical support arms
- suspended nodal tensions
- structures that feel half anatomical, half mathematical

### Circle doctrine
A circle is not banned.
A circle is sacred punctuation.

Use circles only when they serve as:
- regional anchor
- rare compositional event
- contrast against more unstable geometry
- large quiet center for surrounding complexity

---

## 3. Macro-Structure Requirements

The screen needs stronger large-scale composition.

## Minimum required layers

### Layer A — Background atmosphere
- dark gradient or pressure wash
- extremely soft variation
- subtle regional color temperature differences

### Layer B — Macro ghost field
- large field-scale structures
- low-opacity, slow-moving, nonliteral
- must not read as enlarged normal motifs

### Layer C — Mid-scale ecologies
- the main layer of local relationships
- where most eddies and pockets become visible
- most important compositional layer

### Layer D — Accent events
- sparse brighter structures
- higher contrast but still elegant
- should be compositionally rare

### Layer E — Fine particulate support
- tiny residual dots, punctures, soft debris
- only to support scale and atmosphere
- never the main attraction

## Macro field rules
- some macro forms should begin offscreen
- some should exit offscreen
- some should arc across multiple regions
- some regions should be nearly empty but visibly shaped by macro pressure
- not all emptiness should feel neutral

---

## 4. Regional Ecology System

Each region of the screen should have its own personality.

## Each region should influence:
- motif family preference
- closure tendency
- elongation tendency
- fragmentation tendency
- brightness range
- motion tempo
- swirl vs drift bias
- local density
- color bias
- deformation aggressiveness

## Example dialects
You do not need these exact names in code, but the concept should exist.

### Region dialect examples

#### Drift-rib region
- elongated forms
- spine-heavy motifs
- directional bias
- low closure
- slow drift

#### Hollow-shell region
- sparse partial enclosures
- large gaps
- gentle breathing opacity
- medium scale

#### Puncture-cloud region
- micro clusters
- small detached nodes
- scattered short fragments
- slightly higher sparkle density

#### Curl-basin region
- local swirl
- bent arcs
- grouped motion
- nested incomplete shells
- delayed phase coherence

#### Pressure-front region
- repeated directional bends
- flattened fragments
- compression lanes
- stronger macro presence

### Rule
No region should feel like just “same field, different coordinates.”

---

## 5. Motion and Morph Behavior

Motion is still not expressive enough unless it visibly reveals the field.

## Required motion qualities
- slow overall tempo
- local inertia
- regional coherence
- uneven internal reorganization
- temporary synchrony
- delayed settling
- occasional accent emergence

## Morph rule
Motifs should not feel like:
“shape A tweened into shape B”

They should feel like:
“a structure reorganized under pressure”

## Implementation guidance
When morphing:
- do not let all subparts move at the same rate
- assign different lag values to different motif subcomponents
- keep some anchors persistent during morph
- let local satellites drift differently from main shell/spine structures
- preserve tension asymmetry during transitions

## Visible field behaviors to implement
- motifs tangentially align to local flow
- motifs compress in convergence zones
- motifs loosen in quiet basins
- clusters dim/brighten with mild shared rhythm
- some structures arc around invisible centers
- some structures drift into temporary coherence then decay

---

## 6. Composition Tuning

The project needs stronger compositional intelligence.

## Scale hierarchy
There must be a clear difference between:
- macro ghost structures
- mid-scale primary motifs
- small satellites
- fine particles

Right now the field still risks flattening into “many medium-small things.”

## Spatial tension
Introduce regions that feel:
- crowded
- sparse
- pressurized
- calm
- magnetized
- dissolving

## Local neighborhood rules
In some neighborhoods:
- motifs should echo one another
- brightness should lightly correlate
- angle families should relate
- deformation patterns should rhyme
- one motif may appear as a derivative of another

## Offscreen composition
At least some large structures should be only partially visible.
This increases scale and mystery.

---

## 7. Palette and Contrast Guidance

The palette direction is working. Keep it dark and restrained.

## Keep
- deep atmospheric background
- soft neon-adjacent accents
- purple/teal/cyan territory
- rare luminous nodes

## Improve
- selective contrast
- better separation between macro, mid, and accent layers
- slightly clearer line presence in chosen motifs
- occasional small, meaningful glow blooms

## Avoid
- global brightness increase
- everything glowing equally
- cyberpunk poster energy
- flattening all motifs to one faint value

## Rule
Most of the scene should remain soft.
But a few things must be readable enough to anchor the eye.

## 8. Engineering Changes Needed

Below are implementation-level changes likely necessary to actually achieve the visual target.

## A. Expand motif system
Add more motif families that are explicitly:
- asymmetric
- incomplete
- deformation-friendly
- resistant to default circle output

## B. Add post-generation style filters
After motif generation, reject outputs that are too:
- circular
- symmetric
- literal
- icon-like
- orbit-like

## C. Separate macro-field generator from motif generator
Do not reuse normal motif logic for macro forms.

## D. Strengthen region influence
Regions need to affect both:
- composition
- morphology

Not just spawn density.

## E. Increase shared local behavior
Nearby motifs should sample:
- common attractors
- common phase offsets
- common deformation bias
- common brightness envelopes

## F. Add field debug modes
Add toggles or developer overlays to inspect:
- region boundaries / influences
- flow vectors
- attractor zones
- density map
- motif family selection
- closure bias
- scale layer occupancy

## G. Add stronger parameterization
Expose parameters for tuning:
- perfect closure penalty
- asymmetry bias
- regional dialect strength
- macro field density
- accent rarity
- cluster cohesion
- swirl strength
- quiet basin strength
- deformation lag
- shell vs spine family balance

# Part III — Claude-Facing Direct Prompt

Use or adapt the following block directly with Claude.

---

You are now working on the next serious visual pass of the Living Field project.

The current version is promising, but it is still too dependent on circles, arcs, orbit-like motifs, and readable symbolic fragments. It risks becoming beautiful sci-fi UI residue instead of the intended alien geometric field.

Your next task is to push the system decisively toward:

- alien geometric weather
- regional ecologies
- field-shaped emergence
- asymmetrical and incomplete forms
- macro environmental structure
- visible eddies, pockets, and basins
- elegant but uncanny symbolic life

## Non-negotiable corrections

1. Break the circle/orbit dependency.
   - Perfect circles should be rare.

   - Structures should feel produced by shared regional forces, not scattered as independent symbols.

- Add several new motif families that explicitly avoid radial and bilateral comfort.
- Add style filters or rejection logic that suppress too-circular, too-symmetric, too-iconic results.
- Introduce a dedicated macro-form generator.
- Strengthen region dialects so different screen areas bias different shape families and motion behaviors.
- Add subpart lag and uneven morph response so motifs reorganize under pressure instead of tweening mechanically.
- Increase local group coherence.
- Keep the palette dark and restrained, but improve selective readability and layer separation.

## Visual doctrine

The field should feel like:
- symbolic weather
- mathematical organisms
- elegant unknown instruments
- field pressures becoming visible
- a coherent alien ecology

It should not feel like:
- HUD debris
- orbital widgets
- glowy dashboard remnants
- a procedural icon wallpaper

The next pass succeeds if:
- circles are no longer the dominant read
- the screen has visible macro-structure
- some regions clearly feel like distinct ecologies
- motifs are stranger and less predictable
- the field feels unified
- local eddies and pockets are legible
- the result is elegant, atmospheric, and alive

Do not play it safe.
Push the geometry language into a stranger, more memorable place while keeping the system beautiful and coherent.
