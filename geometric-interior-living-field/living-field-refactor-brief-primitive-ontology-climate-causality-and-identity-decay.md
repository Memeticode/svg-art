# Living Field — Refactor Brief: Primitive Ontology, Climate Causality, and Identity Decay

This document is intended to be handed directly to Claude as the next implementation brief.

The project is improving, but the changes are still arriving more strongly in the rendered output than in the underlying ontology. That means the visuals are advancing faster than the engine’s internal beliefs.

Right now, the project still appears to carry a contradiction:

- the art direction says **no circles**
- the runtime and primitive grammar still appear to think in circle-native terms

That contradiction is now the main bottleneck.

The next enhancement pass should stop being an aesthetic nudge and become a targeted structural refactor.

The mission is:

> remove circle-native ontology, turn macro strands into climate, make motifs reveal their climate history, and increase identity decay so the field can no longer settle into comfortable symbols.

---

## Executive Summary

The next stage should focus on five tightly related goals:

### 1. Refactor the primitive grammar
The system must stop treating circles and rings as first-class geometry.

### 2. Delete the old motif vocabulary
Halo/orbit/ring language should be removed from active family and preset logic.

### 3. Redesign the macro layer
The large strands should become climate traces rather than visible rails.

### 4. Strengthen climate causality
Motifs should visibly show the environmental forces that have shaped them.

### 5. Increase identity decay
Readable, stable silhouettes should decay faster and reorganize more continuously.

These five changes should be treated as one coherent move, not five unrelated improvements.

---

# Part I — Why This Pass Matters

The project has already gained real strength:
- the field has atmospheric presence
- the shape language is more distinctive
- macro structure is finally helping
- the scene increasingly feels like climate plus inhabitants

But the project is still held back by three structural problems:

## 1. The primitive ontology still appears to be circle-native
If the lowest layer still prefers circles, rings, orbital accents, or round enclosure logic, then all higher-level anti-circle rules are fighting the substrate instead of being supported by it.

## 2. The macro field is useful but too explicit
The large environmental strands are doing work, but they are close to becoming visible compositional rails. The system needs climate traces, not luminous scaffolding.

## 3. Motifs still hold nameable identity too long
Some forms remain readable as persistent units for longer than they should. The field needs stronger identity half-life and more pressure-shaped reorganization.

The current system still seems to encode circle-first assumptions at the primitive level.

### Action
Replace the current primitive ontology with a path-native, incomplete, non-round ontology.

### Required outcomes
- no dedicated circle slots
- no dedicated ring slot
- no orbit/halo semantic role

- shell fragment

- spine segment

- puncture mark

The primitive layer should encode articulate incompletion by default.

---

## Strike 5 — Increase identity decay

### Problem
Some motifs remain recognizable and stable for too long.

### Action
Shorten silhouette half-life and strengthen continuous reorganization.

### Required direction
- readable silhouettes should destabilize sooner
- asynchronous subpart drift should increase
- symmetry comfort should decay faster
- climate memory should prevent reversion to easy forms
- motifs should betray themselves before they become icons

### Rule
If a motif remains nameable for more than a short interval, it should begin to fail its own completion.

# Part III — Structural Refactor Guidance

This section describes what Claude should actually do.

## 1. Refactor the primitive model into non-round contour logic

### Goal
Replace a circle/ring-oriented primitive vocabulary with a path-native climate-friendly one.

### Conceptual shift
Move from:
- paths plus circles plus ring

Toward:
- contour segments
- shell fragments
- spine segments
- scaffold arms
- puncture marks
- lobe traces
- manifold residues
- climate scars

### Key properties of the new primitive set
- incomplete by default
- asymmetry-friendly
- good under deformation
- no radial comfort state
- capable of building both inhabitants and climate traces
- supports overlap and off-screen continuation naturally

### Rule
The primitive model should not know what a circle is as a preferred form.

---

## 2. Refactor the SVG pool and renderer around path-native elements

### Goal
Make the runtime consistent with the no-circle doctrine.

### Required changes
- stop preallocating `<circle>` elements
- stop preallocating a ring path for every agent
- convert all pooled visual elements to path or polyline-like representations
- support tiny puncture marks as miniature path glyphs or short contour fragments, not circles

### Desired outcome
The runtime should no longer carry hidden circular ontology.

### Rule
If the pool is still circle-native, the system is still circle-native.

---

## 3. Rebuild motif families around climate-compatible geometry

### Strong family set
Use a compact but expressive family vocabulary built around:

#### Broken shells
- asymmetric enclosure remnants
- climate-compatible collapse behavior
- no round rest states

#### Split lobes
- unequal cohabitation
- side-dominance drift
- anti-symmetry

#### Ribbed spines
- directional logic
- lane compatibility
- memory-rich deformation

#### Pressure fragments
- repeated directional residues
- local coherence and decay

#### Asymmetric scaffolds
- unresolved structural organisms
- weighted armature logic

#### Bent manifolds
- environmental continuity
- macro-to-mid scale bridge family

#### Climate scars
- low-opacity residues
- excellent for overlap and time-memory

### Rule
Families should survive distortion without falling back into familiar iconography.

---

## 4. Move anti-circle and anti-comfort logic earlier in the pipeline

### Problem
If invalid forms are generated first and then corrected later, the engine is still thinking incorrectly.

### Better pipeline
1. generate from non-round family grammar
2. evaluate for symbolic comfort early
3. reject or perturb before visibility
4. update continuously under climate and memory
5. use anti-stability logic as a final correction layer

### Add or strengthen scores such as
- closure comfort
- symbolic readability
- silhouette persistence
- radial drift risk
- macro/micro mismatch
- comfort-state risk

### Rule
The engine should want the right geometry before policing begins.

---

### Every motif should sample and accumulate
- flow direction
- lane strength
- basin depth
- curl intensity
- front pressure

- macro trace proximity

### Use those values to drive
- dominant axis drift
- shell opening side
- scaffold lean
- puncture migration
- fragment spacing
- lobe tension
- rib distribution
- contour fracture
- opacity rhythm
- reconstruction vs dissolution tendency

### Rule
Climate is not backdrop. Climate is authorship.

---

## 6. Add stronger structural memory

### Goal
Motifs should feel weathered, not merely animated.

### Suggested memory channels
- accumulated asymmetry
- shell collapse memory
- lane skew memory
- front exposure memory
- curl exposure memory
- puncture migration memory
- contour fatigue
- persistence age
- readability drift
- dominant force history
- climate scar intensity

### Behavior
These values should:
- accumulate gradually
- decay slowly
- bias future reorganization
- prevent clean regression into easy forms

### Rule
The past should remain legible inside the present form.

# Part IV — Macro Layer Redesign Brief

This is likely the fastest route to the next visible leap.

## Current problem
The macro field currently helps but over-signals itself.

## Desired macro feel
Macro structure should feel like:
- field weather
- pressure veils
- contour scars
- basin rims
- climate fronts
- manifold weather
- softened environmental residues

## Redesign instructions

### A. Reduce rail confidence
Avoid long uniform strands with stable brightness and width.

### B. Break and stagger continuity
Macro forms should:
- fade in/out
- vary thickness
- split and recombine
- lose confidence over length
- travel imperfectly
- fragment into related sibling traces

### C. Make macro traces locally consequential
Nearby motifs should visibly inherit:
- directional skew
- shell opening bias
- scaffold alignment
- opacity rhythm
- fragment spacing tendency
- contour deformation pressure

### D. Support off-screen continuation
Major climate structures should belong to a larger system beyond the frame.

### Rule
Macro forms should feel discovered, not drawn.
